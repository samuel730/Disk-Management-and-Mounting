# Disk-Management-and-Mounting
## 📌 Introduction

Disk management is a fundamental skill in Linux system administration and DevOps engineering. It involves creating, organizing, and managing storage devices to ensure efficient data handling and system performance. This project focuses on understanding how Linux handles disks, partitions, filesystems, and mounting processes using practical command-line tools.

In this lab, a virtual disk is created using a loop device in a WSL (Windows Subsystem for Linux) environment. The disk is then partitioned, formatted with the ext4 filesystem, and mounted to a directory for usage. The project also demonstrates how to verify storage configurations and prepare partitions for persistent mounting using `/etc/fstab`.

By completing this exercise, you gain hands-on experience with real-world Linux storage operations commonly performed in DevOps and cloud environments.
